SQUASH MATCH

SPORTS APP, 2015

About

Squash Match is a project to build an app that helps you connect to users and find available sports facilities in your local area. In big cities like London where a large number of people are time poor arranging ad-hoc sports events is nearly impossible. Sadly this has led many over 30's to take up jogging. The aim of Squash Match is to allow users to find others nearby, book nearby courts and play.

How it Works

Users log on to Squash Match and can either search for and book an available court or search for other users who currently have courts booked and are looking for a suitable challenge.

Rating System

One of the biggest factors to consider is the competition. No one want to win or lose 10 - nil. Squash Match uses the Elo rating system to rank players. All players start with 1000 points which are added and deducted for wins and losses. The greater the point difference between the players the more points the winner will get. The algorithm was designed to rank chess players and is still used today. When more testing has been done the algorithm can be tweaked so it more accuratly models the behaviour of sporadic amateur athletes.
